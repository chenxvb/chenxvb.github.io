主要记录一下 pyinstxtractor-ng 和 pycdc 工具的使用

简单看看发现是 Pyinstaller 打包的应用，可以使用 pyinstxtractor-ng 解包。他们有在线服务 https://pyinstxtractor-web.netlify.app/

解包出来有 .so 和 pyc 文件

使用传统的 uncompyle6 不太能转换 pyc，翻了翻工具发现 pycdc 这个工具非常好用。搭好编译环境后写脚本反 pyc

部分脚本会一直卡着，直接 ctrl c 退出。还有些不能正常反汇编。比如核心的 zqdm.py

在初始化时部分函数无法反汇编完成，但是这个可以用 python 汇编人肉看
